### Day 7（2026-01-06） -  语义化标签

- **知识点**："语义"就是"意思"的意思, 这些标签不光显示内容, 还告诉浏览器和搜索引擎: "这一块是页头" "这一块是导航" "这一块是主要内容" "这一块是页脚".
    常见的语义化标签:
    - `<header>`: 页面或区块的头部, 通常放 logo 标题 导航.
    - `<nav>`: 导航栏, 放链接菜单
    - `<main>`: 主要内容区域 (一个页面只能有一个).
    - `<section>`: 一个主题区块, 比如"关于我" "我的作品". 
    - `<article>`: 独立的内容文章 (可选).
    - `<aside>`: 侧边栏, 比如广告, 相关链接.
    - `<footer>`: 页脚, 通常放版权, 练习方式等
- **练习任务**：用语义化标签重构 (改造) 你的个人主页:
    1. 用完整 HTML 结构。
    2. 把页面分成清晰的区块：
        - 用 `<header>` 放大标题和导航链接（可以链接到你之前的每日作业）
        - 用 `<main>` 包裹主要内容
        - 用几个 `<section>` 分隔“关于我”“爱好”“学习进度表”“美女图片”等部分（记得给每个 section 加 id）
    3. 原来的内容（图片、表格、列表）可以直接复制进去.
- **完成情况**：✅ 已完成（文件：day7-my.html）
- **心得**： 学习了一个 VSCode的向上移动单行/多行代码的快捷键, 重构了代码, 补充了一些学习进度信息.

### Day 9（2026-01-08） -  表单标签

- **知识点**：今天我们来学“表单”（form），它就像一个对话框，能让访客填写信息（比如姓名、留言、选择选项），然后提交给你。
    - `<form>` : 整个表单的容器
    - `<input>` : 输入框 (根据 type 属性的不同, 可以是文字, 密码, 按钮等.)
    - `<label>` : 给输入框加说明文字 (让页面更友好)
    - `<textarea>` : 多行文字输入 (适合留言)
    - `<button>` 或 `<input type="submit">`: 提交按钮
- **练习任务**：添加 联系我

### Day 14（2026-01-11） -  CSS 入门阶段

- **知识点**：
    - `background-color: #f0f8ff;`属性设置元素的背景颜色
    - `font-family: "Microsoft YaHei", sans-serif;` 指定字体系列。默认值取决于浏览器。
    - `color: #333;` 文字颜色
    - `margin: 0;` 外边距 - 去掉页面默认的白边
    - `padding: 20px;` 设置内边布局 
    - `text-align: center;` 属性指定元素中文本的水平对齐方式。
    - `list-style: none;` 指定列表样式
    - `display: inline-block;`属性是一个重要的 CSS 属性，用于控制布局。它指定一个 HTML 元素是被视为块级元素还是行内元素。`inline-block`将元素显示为行内级块容器。该元素本身格式化为行内元素，但您可以应用高度、宽度、填充和边距值
    - `text-decoration: none;` 属性添加文本装饰属性
    - `font-weight: bold;`属性设置文本中字符的显示粗细。
    - `a:hover`当鼠标悬停在链接上时选择并样式化它：当你将鼠标悬停在 <p>、<h1> 和 <a> 元素上时，选择并样式化它：
